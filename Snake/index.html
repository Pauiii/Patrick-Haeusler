CosmicSnake

Instructions

  • The game starts if you press 'w', 'a', 's', 'd' or any of the arrow keys.
  • The snake can be controlled with the same keys you start the game.
  • You can pause the game by pressing 'p' or 'pause'.
  • It is impossible to turn the snake by 180 degrees.
  • Every time the snake eats an apple the score will be increased by 9 points.
  • If the snake tries to eat an meteor, or itself or hits the border of the canvas the game will end.
